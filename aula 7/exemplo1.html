Exemplo 1 - Estudo de cores no CSS

As cores no CSS pede ser utilizadas de varias formas como por exemplo

  • As cores podem ser representadas pelo nome, exemplo: red,gren,blue,...
  • As cores podem ser representadas atraves de uma função RGB que e vermelho, verde, azul
  • As cores, tambem pedem ser representadas atraves de um numero em hexadecimal
